# Time.js

## 一、工具函数

```
函数： getCurrentTimeStamp()
功能： 获取现在时间戳
输入： 无
输出： 现在时间戳
事例：
Time.getCurrentTimeStamp()  //1542701124216
```

```
函数： getCurrentFullYear()
功能： 获取现在年份
输入： 无
输出： 现在年份
事例：
Time.getCurrentTimeStamp()  //2018
```

```
函数： getCurrentMonth()
功能： 获取现在月份
输入： 无
输出： 现在月份 1,2,3,4...
事例：
Time.getCurrentMonth()  //11
```
